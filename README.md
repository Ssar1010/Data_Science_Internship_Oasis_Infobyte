# Data Science Virtual Internship Projects

A collection of five data science and machine learning projects completed during a **4–6 week virtual internship program**. The projects cover classification, regression, exploratory data analysis, and natural language processing using Python and popular data science libraries.

## 📌 Projects

### 1. Iris Flower Classification
A classification model that predicts the species of an iris flower — **Setosa, Versicolor, or Virginica** — based on its sepal and petal measurements.

### 2. Unemployment Analysis with Python
An exploratory data analysis project investigating regional and temporal unemployment trends in India, including the impact of the **COVID-19 pandemic**.

### 3. Car Price Prediction
A regression model designed to predict the selling price of used cars based on features such as brand, age, mileage, fuel type, and transmission.

### 4. Email Spam Detection
An NLP-based binary classification system that distinguishes between **spam** and legitimate (**ham**) emails using text processing and machine learning techniques.

### 5. Sales Prediction
A regression model that predicts product sales based on advertising expenditure across **TV, radio, and newspaper** media channels.
